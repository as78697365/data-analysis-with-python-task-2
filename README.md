# Titanic Survival Analysis

## Overview
This project analyzes the Titanic dataset using Python. The objective is to explore survival patterns based on gender, passenger class, and age group.

## Dataset
The dataset used is the Titanic dataset from Kaggle, containing passenger information such as age, gender, class, fare, and survival status.

## Tasks Performed
- Data Loading
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Data Visualization
- Survival Pattern Analysis

## Questions Answered
1. Who survived more: males or females?
2. Did passenger class affect survival chances?
3. What was the survival rate by age group?

## Visualizations
- Survival Rate by Gender
- Survival Rate by Passenger Class
- Survival Rate by Age Group
- Passenger Age Distribution Histogram

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Female passengers had a higher survival rate than male passengers.
- First-class passengers had better survival chances.
- Younger passengers generally showed higher survival rates.
